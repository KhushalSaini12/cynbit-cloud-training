# How a Browser Loads a Website

When a user enters a website URL in a browser, several steps occur before the webpage is displayed.

## 1. URL Processing
The browser reads the URL entered by the user, such as https://www.google.com.

## 2. DNS Lookup
The browser contacts a DNS server to convert the domain name into an IP address.

## 3. Establishing Connection
The browser establishes a TCP/IP connection with the web server. If HTTPS is used, a TLS/SSL handshake is performed.

## 4. Sending HTTP Request
The browser sends an HTTP or HTTPS request to the server.

## 5. Server Response
The server returns HTML, CSS, JavaScript, images, and other resources.

## 6. Rendering the Page
The browser processes the resources and displays the webpage.

## IP Address
An IP address uniquely identifies a device on a network.

## DNS
DNS converts domain names into IP addresses.

## Ports
Port 80 is used for HTTP and Port 443 is used for HTTPS.

## Protocols
Protocols are rules that govern communication. Examples include HTTP, HTTPS, TCP, IP, and DNS.

## Conclusion
A browser loads a website by resolving the domain name, connecting to the server, requesting resources, and rendering the content.
